What is

JOSM ?

JSOM is a free software desktop editing tool for OpenStreetMap geodata created in Java. Originally developed by "Immanuel Scholz" and currently maintained by "Dirk St√∂cker". The editing tool contains advanced features that are not present in OSM's default online editor, iD

The highest number of edits in OSM are done using JOSM. The software was used to perform several large scale OSM imports, including TIGER data in the United States.

Some notable features

  • JOSM are importing GPX files (GPS tracks)
  • Working with aerial imagery (including WMS, TMS and WMTS protocols)
  • Support for multiple cartographic projections, layers, relations editing
  • Data validation tools
  • Data filtering
  • Offline work
  • 3D Viewing
  • There are more than 100 plugins
